# 各种零件尺寸

### 统一的标准:

   1. 壁厚2mm, 内部隔断(例如电池仓)厚度1mm
   2. 所有螺丝柱壁厚1mm ~ 2mm (螺丝直径较大时壁厚大, 直径较小时壁厚小)
   3. 过渡处务必做的圆滑, 切面最好是圆锥曲线
   4. 设计尺寸不要太大, 可以摆放元器件即可
   5. 所有提供的三维尺寸都有富余
